# MissingMigrantsAnalysis
The Missing Migrants Project, initiated by the International Organization for Migration, documents migrants who have gone missing or have died in their migration to an another location. This project analyzes the data by region, incidents, migration route, and the time period from 2014-2023.
## Tools Used
* Python
* Pandas
* Jupyter Notebook
* SQL (SQLite)
* Tableau
## Questions
* Which regions experienced the highest number of migrant fatalities?
* What were the leading causes of death?
* Which migration routes had the highest fatalities?
* How did fatalities change over time?

## Key Findings
* The top 5 regions of incidents were: Mediterranean (9703), Northern Africa (5702), North America (3707), Southern Asia (2272), and Western Africa (2180).
* The main leading cause of death is drowning (12578) followed by unknown cause of death (6561) and Vehicle accident / death linked to hazardous transport (4887).
* The migration route with the highest deaths are Unknown (7412 deaths) followed by Central Mediterranean (7114), Sahara Desert Crossing (5617 deaths), and US-Mexico Border Crossing (4512 deaths). The number of missing migrants however is more frequently documented in the Central Mediterranean route with 14,902 missing.
* Deaths and number of missing fluctuates througout the years with the minimums recorded at 1725 deaths and 1409 missing and maximums recorded at 4296 deaths and 2766 missing.
